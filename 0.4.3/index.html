Overview

Firezone is an open source, self-hosted VPN server and egress firewall for Linux. Use it to quickly and easily secure access to your private network and internal applications from a simple Web UI.

These docs explain how to deploy, configure, and use Firezone.

Quick Start

  1. Deploy: A step-by-step walkthrough of setting up Firezone. Start here if you are new.
  2. Administer: This section relates directly to configuring the server instance.
  3. User Guides: Useful guides to help you learn how to use Firezone and troubleshoot common issues. Consult this section after you successfully deploy the Firezone server.

Common Configuration Guides

  1. Split Tunneling: Only route traffic to certain IP ranges through the VPN.
  2. Whitelisting with VPN: Configure a VPN server with a static IP address.
  3. Reverse Tunnels: Establish tunnels between multiple peers.

Get Help

If you’re looking for help installing, configuring, or using Firezone, we’re happy to help.
